# 八字流年 · 大白话解说（Julie专用 · 温柔版）

这是一个给自己专用的小工具：  
输入生日 → 自动排八字 → 自动看流年 → 自动判断五行偏旺/偏弱 →  
给出水晶、精油、左右手佩戴建议、四种佩戴模式、挂件建议、可直接复制给命主的 WhatsApp 文案。

不花俏、好复制、好理解、手机也能用。

---

## ✅ 功能特色

- 自动排八字（年柱 / 月柱 / 日柱 / 时柱）
- 自动看流年（输入年份即可）
- 自动判断五行偏旺 / 偏弱
- 推荐对应的水晶和精油
- 左右手佩戴建议（贵人/财运、护身/化煞）
- 4 种佩戴模式：日常、开运、灵修、应急
- 支持挂件：钱袋、福袋、五帝钱、貔貅、古币、算盘、莲花牌
- 可直接复制 WhatsApp 文案，一次说清楚、不啰嗦
- 手机可使用（iPhone / iPad / Android）

---

## ✅ 使用方法

1. 打开网页（GitHub Pages 地址）  
2. 输入生日、公历日期、出生时间（不知道可选“未知”）
3. 选时区（马来西亚/新加坡用 UTC+8）
4. 输入想看的流年年份（不填则自动用今年）
5. 点击「生成解说」

页面下方会出现：
- 八字排盘简表
- 自动生成的解说文字
- 按钮「复制解说」可直接贴给命主

---

## ✅ 更新方法（超简单）

- 打开仓库
- 点 `index.html`
- 按右上角铅笔（Edit）
- 修改完成后「Commit changes」
- 刷新网页即可更新
